# Sharing a Git Folder in Databricks Free Edition

## Steps

1. **Add user in workspace settings**
   - Go to (Your logo in the top right corner) → Settings → Identity and Access -> Users (Manage)
   - Add the user's email address (New User)

2. **User accepts invitation**
   - User receives email invitation
   - Clicks link to accept and create account

3. **Share the Git repo IN DATABRICKS (not on github)**
   - Navigate to your home folder
   - Find the Git repo (e.g., `ceu-modern-data-platforms-databrcks`)
   - Click the share icon/option
   - Add the user with **"Can Manage"** permissions

4. **User accesses the repo**
   - User logs into the workspace
   - Finds the repo under **"Shared with me"** section
